### Snake game
 Snake game은 키보드를 사용하여 뱀을 조종해 먹이를 먹고 몸길이를 키우며 점수를 얻는 게임



--- 
### :exclamation: 게임 규칙 :exclamation:

#### :question: 게임 방법 
방향키를 사용하여 플레이 한다.

→ : 오른쪽으로 이동
← : 왼쪽으로 이동
↑ : 위로 이동
↓ : 아래로 이동


#### 점수 획득 방법
- 먹이를 먹으면 점수가 1점씩 오름


#### 게임 종료 :thumbsdown:
- 뱀이 컴퓨터 화면을 벗어났을 때
- 몸의 길이가 늘어나 자신의 몸과 부딪혔을 때
